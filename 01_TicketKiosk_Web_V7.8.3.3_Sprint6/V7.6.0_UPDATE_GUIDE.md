# V7.6.0 Core Engine 覆蓋說明

## 覆蓋方式

將更新包內所有檔案，依原本資料夾結構覆蓋到網站專案根目錄。

本次需要覆蓋：

- `staff.html`
- `service-worker.js`
- `js/core/ticket-status-engine.js`（新增）
- `js/staff/order-center.js`
- `js/staff/ticket-validator.js`
- `CHANGELOG.md`
- `docs/V7.6.0_ENTERPRISE_CORE.md`（新增）
- `tests/ticket-status-engine.test.js`（新增）
- `tests/service-worker-behavior.test.js`

## Firebase

本版不需要修改 Firebase Rules，也不需要部署 Functions。

## 部署後

1. 上傳並覆蓋檔案。
2. 重新部署 Firebase Hosting。
3. 員工手機或平板重新整理頁面一次。
4. 如果仍看到舊畫面，關閉分頁後重新開啟；新 Service Worker 會刪除舊快取。

## 測試流程

1. 建立新訂單。
2. 員工後台查詢訂單並確認入場。
3. 驗票卡確認離場。
4. 關閉驗票卡後重新查詢同一筆訂單。
5. 訂單狀態應顯示「已離場」，不再顯示「遊玩中」。
