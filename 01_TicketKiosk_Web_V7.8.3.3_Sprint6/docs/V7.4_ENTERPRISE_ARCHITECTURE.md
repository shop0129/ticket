# 小怪獸售票機 V7.4 Enterprise

V7.4 採「不破壞 V7.3 功能」的漸進式企業架構。既有售票、Staff、Dashboard、Firebase 與 PWA 模組全部保留，新增統一核心層。

## 新增核心

- `js/core/enterprise-core.js`：版本、事件匯流排、錯誤日誌、系統健康狀態、權限 API。
- `js/core/enterprise-bridge.js`：橋接既有 Role System 與新事件層。
- `js/core/enterprise-guards.js`：依 `data-permission`、`data-role` 自動隱藏介面。

## 權限 API

```js
MonsterEnterprise.can("order.create");
MonsterEnterprise.require("system.settings");
```

店長角色 `admin` 擁有全部權限；員工角色 `staff` 只保留售票、查詢、會員新增與 Dashboard 檢視。
