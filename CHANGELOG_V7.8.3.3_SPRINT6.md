# V7.8.3.3 Sprint 6 更新內容

- 改以 V7.8.3.3 完整正式版重新整合，不再使用 V7.3 網頁底檔。
- 現金按鈕連線本機 127.0.0.1:8765 安全橋接。
- 首次使用輸入8位配對碼，配對資料只保存在點餐機瀏覽器。
- 現金控制器只收到扣除會員點數後的實付金額。
- 全額點數折抵時不開啟收幣或收鈔設備。
- 付款剛好完成後才接受唯一出票授權。
- 相同訂單、付款ID或出票授權重送時，不重複建立訂單、累積會員、更新統計或出票。
- 出票動畫完成後才回覆 Android 控制器「已出票」。
- 保留 V7.8.3.3 的會員點數表單重設、票券資料同步、營業模式、贈品規則、作廢保護、QR驗票及 Firebase 後台同步。
- 中斷恢復維持保守策略；不確定狀態轉人工處理。
- 更新 PWA App Shell、快取版本與首頁版本標記。
- Android MDB 找零、出幣、出鈔、Reset、Escrow及未知指令封鎖維持不變。
