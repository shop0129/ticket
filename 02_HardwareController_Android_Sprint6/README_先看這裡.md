## 先確認硬體分工

- `/dev/ttyS1 @ 9600`：MDB，硬幣機與紙鈔收鈔器共用。
- `/dev/ttyS3 @ 9600`：出鈔找零機，不是收鈔器；Sprint 6 僅供監聽，禁止傳送。
- `/dev/ttyUSB0`：EDC。

## 建議測試順序

1. 關閉原廠 Lite QC Tool。
2. 用 Android Studio 安裝本專案。
3. 開啟 ttyS1 的系統權限後，在 Explorer 選擇 ttyS1、9600 並連線。
4. 記下畫面綠色區塊顯示的8位配對碼。
5. 依 `README_Sprint6_PWA現金橋接.md` 部署新版售票機；第一次按現金付款時輸入配對碼。
6. 測試完成後匯出 Session ZIP。

不要在 Command Library 隨機輸入 HEX，也不要操作原廠工具的取幣、出鈔或 Reset。
