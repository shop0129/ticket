# Sprint 2 · MDB 安全補幣測試

## 安全限制

- 只使用 `/dev/ttyS1`、9600 baud。
- 「開始補幣」依序送出 Coin Setup `09` 與 Enable `0C FF FF FF FF`。
- 60 秒逾時、按停止、App 進入背景或中斷連線時，自動送出 Disable `0C 00 00 00 00`。
- Reset `08`、出幣 `0D`、支付 `0F 02` 與所有未知 MDB 指令均被封鎖。
- 本版沒有「取幣」或「支付」功能。

## 第一次測試

1. 確認標題顯示 `Sprint 2 MDB Coin Decoder`。
2. 選擇 `ttyS1 · MDB`、`9600`，按「連線」。
3. 確認 RX 持續增加，`30 09` 顯示為「紙鈔機停用（正常待機）」 。
4. 按「開始補幣（60秒）」。
5. 先投入一枚 10 元，確認硬幣被收下且畫面出現「NT$10 → 儲幣筒」或「溢幣箱」。
6. 按「停止補幣」，再投入硬幣應被退回。
7. 按「查詢 Tube Info」，確認畫面顯示各面額庫存。
8. 匯出 ZIP；ZIP 內應有 `session.log`、`mdb_summary.txt` 與各 Port Raw 檔。

第一次只測一枚 10 元。不要操作原廠工具的「取幣」、`RESET` 或出鈔功能。
