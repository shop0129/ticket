# v0.5 實機測試

1. 先停止原廠程式並設定 tty 權限。
2. 連線 `/dev/ttyS1` @ 9600，確認 RX 持續收到 `33 30 20 30 39 0D 0A`。
3. 先不要解除 TX 鎖定，確認按鈕無法送出。
4. 僅在確認指令安全後勾選 TX 解鎖；60 秒後應自動上鎖。
5. 加入 Queue 後可傳送下一筆；傳送全部會先二次確認並以 250ms 間隔送出。
6. 送出成功後 TX History 應出現指令；可選取後按「重送 History」。
7. TX 後 1500ms 內收到的 RX，畫面應標記 `RESPONSE→TX #n`。

注意：未知指令可能觸發收款、退幣或出鈔，請勿隨意測試。
