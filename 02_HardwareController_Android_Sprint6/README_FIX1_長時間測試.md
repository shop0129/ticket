# FIX1 長時間測試

1. 安裝 FIX1，確認標題顯示 `FIX1 Performance`。
2. ttyS1 選 9600 並連線。
3. 先持續運行 10 分鐘，不操作畫面。
4. 確認 RX 數量持續增加、Last RX 時間持續更新。
5. 相同封包應折疊顯示為 `REPEATED×次數`，不再逐包增加大量文字。
6. 按「暫停顯示」並等待 5 分鐘；RX 數量與 Last RX 仍應持續更新。
7. 按「繼續顯示」，畫面應在約 0.5 秒內恢復。
8. 再持續運行到 30 分鐘，切換 ttyS1／ttyS3，畫面應保持可操作。
9. 匯出 ZIP，檢查 `session.log`、`mdb_rx_raw.bin` 均有資料。

通過標準：30 分鐘沒有停住、Last RX 持續更新、ZIP 可匯出。
