# MonsterHardwareSDK v0.3

## 新增
- Protocol Analyzer 統計面板
- RX / TX packet counter
- RX / TX byte counter
- RX 封包時間差（Time Delta）
- HEX、ASCII、長度、ASCII/Binary 類型分行顯示
- Session 開始時間、目前 Port、Baud、Last RX
- Log 檔案加入 Session 與統計表頭

## 修正
- 匯出 Log 後顯示「已開啟匯出選單」
- 匯出時加入 ClipData 與 URI 讀取權限
- 無可用分享程式時顯示明確錯誤

## 保留
- TX 安全鎖
- 暫停 / 繼續顯示
- 儲存 Log
- JNI Serial open/read/write 流程不變
