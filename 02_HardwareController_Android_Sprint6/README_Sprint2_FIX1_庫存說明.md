# Sprint 2 FIX1 測試說明

本版依 `explorer_20260721_222608.zip` 實測修正。

## 已確認的實機資料

- `08 52 00`：NT$10 進入儲幣筒；硬幣機同時回報該筒數量為 0。
- `0A` Tube Info：硬幣機回傳 16 個 Coin Type 的數量全部為 0。
- 原廠 Lite QC Tool 在同一台機器上也顯示 NT$1、NT$5、NT$10、NT$50 全部為 0。
- Coin Setup 路由顯示可進 Tube 的面額是 NT$1、NT$5、NT$10、NT$50；NT$20 不屬於 Tube。

因此畫面現在分成：

1. `本次實收`：依實際收幣事件計算，這是本次測試最可信的收幣數。
2. `硬幣機 Tube 回報`：保留裝置原始庫存回覆供診斷；若全為 0，不代表本次沒有收幣。

## 再測一次

1. 關閉 Lite QC Tool。
2. 連線 `ttyS1 · 9600`。
3. 按「開始補幣（60秒）」並只投入一枚 NT$10。
4. 確認顯示 `本次實收明細：NT$10×1`。
5. 停止補幣後按「查詢硬幣機 Tube」。
6. 即使裝置仍回報 0，本次實收仍應保留 NT$10×1。
7. 匯出 ZIP 供後續分析。
