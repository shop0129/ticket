## 主要入口
- `index.html`：Kiosk 售票機
- `staff.html`：Staff／店長後台
- `lobby-display.html`：大廳顯示
- `play-display.html`：遊玩區顯示

## 核心資料夾
- `js/`：系統程式
- `css/`：介面樣式
- `images/`：票券與 PWA 圖片
- `sounds/`：操作音效
- `tests/`：自動測試
- `docs/`：架構與維護文件

## 部署
