# V7.6.1 Validation Center 2.0 更新說明

## 更新內容
- 員工端新增等待入場、遊玩中、已超時、今日已離場即時分類。
- 新增今日現場清單，點選訂單即可直接開啟驗票卡。
- 驗票卡新增預定離場倒數與超時顯示。
- QR 掃描或短訂單號查詢後直接開啟驗票卡。
- 入場與離場後即時刷新分類數量與現場清單。

## 覆蓋方式
將更新包內容依原資料夾位置覆蓋，再重新部署 Firebase Hosting。
本版不需修改 Firebase Rules，也不需部署 Functions。

## 測試
1. 建立一筆等待入場訂單，確認「等待入場」數量增加。
2. 點選該訂單完成入場，確認移到「遊玩中」。
3. 調整或等待預定離場時間，確認超時後移到「已超時」。
4. 完成離場，確認移到「今日已離場」。
