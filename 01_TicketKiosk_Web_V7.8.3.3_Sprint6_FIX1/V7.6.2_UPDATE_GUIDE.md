# V7.6.2 Live Timer 更新說明

## 覆蓋檔案

將更新包解壓縮後，依照原資料夾結構直接覆蓋：

- `staff.html`
- `css/staff.css`
- `js/core/live-timer-engine.js`
- `js/staff/ticket-validator.js`
- `service-worker.js`
- `CHANGELOG.md`

## 部署

1. 覆蓋檔案。
2. 重新部署 Firebase Hosting。
3. 關閉員工端舊分頁或 PWA。
4. 重新開啟；若仍看到舊畫面，清除網站快取後再開。

本版不需要更新 Firebase Rules，也不需要部署 Functions。

## 測試

1. 讓一筆訂單完成入場，確認清單每秒倒數。
2. 將預定離場時間調到 10 分鐘內，確認顯示橘色「即將到時」。
3. 超過預定離場時間，確認自動移到「已超時」。
4. 完成離場後，確認倒數停止並移到「今日已離場」。
5. 將頁面切到背景再切回，確認倒數依照實際時間續算。
