# V7.5.1 訂單 QR 與短訂單號碼

本階段建立後續驗票系統的訂單識別核心。

- 每筆新訂單自動產生短訂單號碼，例如 `A01238`
- 每筆訂單產生隨機 QR 驗證碼與 `MGV1` QR payload
- 每張入場票建立獨立 `ticketInstanceId`
- 成功付款畫面顯示可列印的收據預覽與 QR Code
- Staff 後台新增快速驗票入口、今日訂單捷徑及短碼查詢
- 相機掃描會在 V7.5.2 啟用；本版已先完成可供掃描器使用的資料格式

QR Code 不包含價格與完整訂單內容，只包含訂單識別與隨機驗證碼。掃描後必須向 Firebase 取得最新訂單資料。
