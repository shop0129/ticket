# V7.6.2.1 Receipt & Mobile Scanner Hotfix

## 修正內容
- iPhone Safari、部分 Android Chrome 沒有 `BarcodeDetector` 時，改用 html5-qrcode 相機掃描備援。
- 保留原生 BarcodeDetector 快速掃描路徑。
- 顯示 HTTPS、外部瀏覽器與相機權限提示。
- 相機無法啟動時，可直接改用訂單號碼。
- 收據依 58mm 熱感紙安全寬度排版，避免文字與金額超出框線。
- QR Code 固定為 28mm，列印時移除預覽虛線框。

## 覆蓋與部署
1. 解壓縮更新包並依原資料夾位置覆蓋。
2. 重新部署 Firebase Hosting。
3. 手機關閉舊分頁後，以 Safari 或 Chrome 重新開啟。
4. 第一次掃描時允許相機權限。

不需要更新 Firebase Rules 或 Functions。

## 注意
在沒有原生 BarcodeDetector 的瀏覽器，第一次掃描需連網載入備援掃描元件。若從 LINE 內建瀏覽器開啟，請改用外部 Safari/Chrome。
